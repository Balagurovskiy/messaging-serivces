# messaging-serivces

1. Имплементация 2х сервисов: 1. Сервис1 -генерирует много сообщений,  
2. Сервис 2 - принимает, обрабатывает и записывает в Н2, сервис-2 - сервис с секьюрити - обязательно. 
логирование отдельным компонентом.
